# Medical Assistant

AI-powered medical assistant for healthcare professionals with advanced diagnostic support and patient interaction capabilities.

## Features

- **Embedded Interface**: Seamlessly integrated within the portal
- **AI Diagnostics**: Advanced diagnostic support tools
- **Patient Interaction**: Interactive patient care features
- **Secure Access**: Authenticated access through portal
- **External Link**: Option to open in new tab if needed

## How to Use

1. **Access**: Navigate to the Medical Assistant from the healthcare section
2. **Interface**: Use the embedded interface directly within the portal
3. **External Option**: Click "Open in New Tab" if you prefer a separate window
4. **Authentication**: Your portal authentication carries over

## Technical Details

- **Integration**: Embedded iframe with secure sandbox
- **Authentication**: Portal-based authentication
- **Responsive**: Adapts to different screen sizes
- **Security**: Sandboxed iframe with appropriate permissions

## Permissions

The embedded application has access to:
- Camera (for medical imaging)
- Microphone (for voice interactions)
- Secure forms and popups
- Same-origin requests
